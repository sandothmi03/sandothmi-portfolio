GUI Horse Race Management System using Java

This project is a comprehensive Java-based GUI application designed to manage a horse race event called "Rapid Run."1 The system leverages JavaFX for the user interface and incorporates Object-Oriented Programming (OOP) principles to ensure robust and maintainable code.
Key features include:
Horse Management: Add, update, delete, and view horse details, including ID, name, jockey, age, breed, race record, and image.
Race Simulation: Randomly select horses for the race, assign race times, and display winning horses with their times.
Data Handling: Save and retrieve horse details from text files, ensuring data persistence and integrity.
Visualization: Graphically display race results and horse performance.
This project demonstrates proficiency in algorithmic problem-solving, GUI design, and OOP concepts such as encapsulation and inheritance.
