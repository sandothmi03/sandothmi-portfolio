Path Finding Robot using DS&A

As part of my coursework on Data Structures and Algorithms, I developed a Java program that enables an autonomous robot to navigate a grid-based environment filled with obstacles. The main objective was to implement efficient data structures and algorithms for path planning, ensuring that the robot finds the most optimal path while avoiding obstacles.
My responsibilities included designing custom data structures such as grids and robot states, and implementing an efficient pathfinding algorithm. Additionally, I integrated an obstacle detection mechanism and simulated the robot's movement across different grid configurations to test the functionality. This project enhanced my skills in algorithmic thinking, complexity analysis, and customized data structure implementation for solving real-world problems.
