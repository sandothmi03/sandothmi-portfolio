Fashion, Makeup and Beauty Website

For my Web Technology coursework, I participated in the collaborative development of a multi-functional web application. The project involved using web technologies such as HTML, CSS, JavaScript, and XML to create a fully integrated website with various interactive features.
As the designated Student 3, my primary responsibility was the implementation of the Product Purchase Page. This included designing an intuitive user interface where users could select and order products. I used HTML and CSS to structure and style the page, ensuring it adhered to modern UI/UX principles. JavaScript was implemented to provide dynamic functionality, such as automatic price calculation based on product quantities and discount points obtained from a quiz feature. I also developed the About Us Page, which featured a mouse-over effect that dynamically displayed the details of each team member. This enhanced user engagement and provided a personal touch to the website. I ensured that the page was responsive, accessible, and consistent with the overall website design.
The project taught me the importance of collaboration, web accessibility, and user-centered design while giving me hands-on experience with form validation, CSS styling, and JavaScript interactivity.
