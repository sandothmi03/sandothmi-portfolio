CLI Horse Race Management System using Python

This project is a command-line application developed as part of a programming fundamentals coursework. It was designed to manage horse race events, specifically for an event called 'Rapid Run.' The system allows event organizers to register, update, delete, and view horse details. Key features include sorting horses by their breed, saving details to a text file, and randomly selecting four horses for the final round. It also simulates a final race by assigning random times to the horses and displays the top winners. The project utilizes Python and focuses on file handling, decision-making, and loops, reinforcing object-oriented design principles and algorithmic problem-solving skills.
